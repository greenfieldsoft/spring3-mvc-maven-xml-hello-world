Maven - Spring 3 MVC Hello World to TEST CI
===============================
Template for Spring 3 MVC + JSP view + XML configuration, using Maven build tool.

### Technologies used
* Maven 3
* Spring 3.2.13.RELEASE
* JSTL 1.2
* Logback 1.1.3
* Boostrap 3
